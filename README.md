# rock-paper-scissors

This is very simple game, rock paper scissors. You will have to play it in the 
console. 

To open the console,
* Open the live page link in a pc, laptop.
* Right-click anywhere on the page.
* Click on inspect.
* Go the console tab.

The messages will be displayed here.

### How to play the game

- The game will have 5 rounds in total.
- You must enter rock, paper or scissor in the prompt.(no need to be case sensitive)
- The computer will make decisions randomly.
- At the end the game will announce the winner.
- To restart just refresh the page.

#### NOTE:
If the prompts start immediately after you open the web page then just answer them and 
then open the console and refresh the game again.
